# dgv
Simple Movie Searching App using Naver Movie API

## Features
* Movie Search using RxSwift
* Movie Search Detail Page using Webkit
* REST API - Naver Movie API

## Technology Stack
`Swift`, `Almofire`, `RxSwift/RxCocoa`
